# Vue Vite Todo App

## Getting Started

### Prerequisites

Make sure you have the following installed:

- Node.js (version 14 or later)
- npm

### Installation

1. Clone the repository:

   ```bash
   git clone https://github.com/maxKharlanov501/NuxGame
   cd NuxGame
  
  
2. Install dependencies:

  ```bash
  npm install
  npm run dev


3. Deployment:

  ```bash
  npm run build
